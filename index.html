Fast static site

This is a minimal, accessible static site ready for deployment to GitHub Pages. A sitemap and robots.txt are included so search engines can index the content. Add the Google Search Console verification meta tag (or HTML file) as described in the README to verify ownership.

Publications

This site hosts my published research papers. Each paper page includes citation metadata (Highwire-style meta tags and JSON-LD) and links to the full-text PDF so Google Scholar and other scholarly services can discover them. Visit Papers to see the list.
